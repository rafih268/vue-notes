## Installation
1. Clone the repository:
```bash
git clone https://github.com/rafih268/vue-notes.git
```

2. Install dependencies:
Go to the root folder and run the following:
```bash
npm install
```

## Usage
To run the project, use the following command:
```bash
npm run dev
```
